# Banco de Dados — Dissertação de Mestrado

Banco relacional em SQLite construído a partir de dados reais da dissertação:
**"Trabalho Docente no Contexto da Reforma do Ensino Médio: reflexões, práticas e relatos em duas escolas de Araçatuba-SP (2022-2024)"**
Autor: Gustavo Aparecido de Paula — ProfSocio/UEL (2024)

## Tabelas
- `escola` — 4 escolas estaduais de Araçatuba-SP
- `disciplina` — 20 disciplinas do currículo paulista
- `grade_curricular` — matrizes curriculares homologadas 2024
- `aula_grade` — carga horária por disciplina e série
- `indicador_educacional` — dados IBGE sobre Brasil, SP e Araçatuba
- `categoria_docente` — categorias do magistério público paulista (Quadro 1)

## Tecnologias
SQLite • DBeaver • SQL
